Mi último proyecto

Mi inteligencia artifical simulando un psicólogo para mejorar la salud mental.

Para el desarrollo de este proyecto, se implementó la técnica de fine-tuning utilizando el modelo GPT-2 de HuggingFaces con 1.5 millones de parámetros. Se establecieron los siguientes hiperparámetros: 12 épocas de entrenamiento, un tamaño de lote de 64 y una tasa de aprendizaje de 1e-4.

La base de datos utilizada para entrenar el modelo constó de 10,000 conversaciones públicas entre psicólogos y pacientes. Estos datos permitieron al modelo adquirir conocimientos y patrones relacionados con la interacción y las dinámicas de las sesiones terapéuticas.

Además dispone de capacidad para traducir mensajes a cualquier idioma. Esto se logró gracias al uso de bibliotecas de Python específicas que permitieron realizar la traducción de manera eficiente.
