Desarrollé mi primera Convolutional Neural Network (CNN) multi etiqueta utilizando la base de datos de Toxic Comments.

Para hacer accesible esta herramienta, la integré en una aplicación web utilizando Streamlit.

La aplicación permite a los usuarios cargar comentarios y obtener predicciones sobre su toxicidad en varias categorías.

Este proyecto se realizó para mejorar mis habilidades en deep learning.
